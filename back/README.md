# Backend - API Express

API REST développée avec Express.js et Airtable.

## Installation

```bash
npm install
```

## Développement

```bash
node server.js
```

L'API sera accessible sur http://localhost:3000

## Technologies

- Node.js
- Express
- Airtable API
- JWT
- bcryptjs
- Zod

## Variables d'environnement

Voir le fichier `.env` à la racine du projet.
