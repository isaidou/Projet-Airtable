# Frontend - Portfolio ESGI

Application React développée avec Vite.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

L'application sera accessible sur http://localhost:2000

## Build

```bash
npm run build
```

## Technologies

- React 19
- Vite 6
- React Router DOM
- Tailwind CSS
- Zod
- React Hook Form
